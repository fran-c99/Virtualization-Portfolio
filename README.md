# Virtualization

## Introduction

A Virtual Machine (VM) is an emulated computer where0 we can run different Operating Systems and apps with virtualized resources. Since a Virtual Machine is isolated from the rest of the system, the VM can’t, in theory, interfere with the host computer's primary OS. This allows users to perform secure tasks like software development and testing or using different operating systems while optimizing physical hardware.
